# Bonjou v1.0.0

First release of Bonjou.

## Features

- Send messages to people on your local network
- Send files and folders
- Works on Mac, Linux, and Windows
- No internet needed, works offline on LAN
- Simple @ commands

## Download

| Platform | File |
|----------|------|
| Mac | bonjou-macos.tar.gz |
| Linux | bonjou-linux.tar.gz |
| Windows | bonjou-windows.zip |

Download from [GitHub Releases](https://github.com/hamzaabdulwahab/bonjou-cli/releases/tag/v1.0.0).

## Install

See [install guide](../install-guide.md) for full instructions.

Quick install:

**Mac:**
```bash
brew tap hamzaabdulwahab/bonjou https://github.com/hamzaabdulwahab/homebrew-bonjou
brew install bonjou
```

**Windows:**
```powershell
scoop bucket add bonjou https://github.com/hamzaabdulwahab/scoop-bonjou
scoop install bonjou
```

**Linux:**
```bash
wget https://github.com/hamzaabdulwahab/bonjou-cli/releases/download/v1.0.0/bonjou_1.0.0_amd64.deb
sudo dpkg -i bonjou_1.0.0_amd64.deb
```
